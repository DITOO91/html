2 FRAME WORK CSS YANG POPULER SAAT INI

Desain front-end menjadi bagian yang sangat penting di website atau aplikasi. Itulah kenapa menguasai bahasa pemrograman CSS diperlukan. Namun, kini Anda bisa lebih mudah melakukannya dengan CSS framework. Dengan framework ini, Anda lebih cepat untuk menciptakan desain front-end tanpa harus coding dari nol. Ada banyak sekali pilihan framework CSS yang dengan fitur dan elemen yang berbeda. Supaya Anda bisa menemukan framework yang tepat, berikut kami jabarkan 2 framework CSS yang paling populer di kalangan developer. Mari kita simak!

Bootsrap

Bootstrap adalah framework HTML, CSS, dan JavaScript yang berfungsi untuk mendesain website responsive dengan cepat dan mudah.  Framework open source ini diciptakan pada tahun 2011 oleh Mark Otto dan Jacob Thornton dari Twitter. Itulah kenapa dulunya Bootstrap dinamakan Twitter Blueprint.  Bootstrap dengan cepat meraih popularitas digunakan oleh 27% website di seluruh dunia. Hal itu karena kesederhanaan dan konsistensi yang ditawarkan Bootstrap dibanding framework lainnya saat itu.

Tailwind

Tailwind css adalah sebuah framework css yang sedang hype di kalangan web developer. Tidak seperti kebanyakan framework css lainnya, tailwindcss tidak terlalu banyak ikut campur dalam penamaan class cssnya yang memungkinkan kita untuk mencopot dan memasang setiap class pada tiap komponen sesuai kebutuhan, atau istilah kerennya utility based class.
